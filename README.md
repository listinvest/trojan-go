# Trojan-Go

A Golang version of [Trojan](https://github.com/trojan-gfw/trojan)

## Differences between Trojan and Trojan-Go

| | Trojan | Trojan-Go |
|------|--------|-----------|
| Language | C++ | Golang |
| MySQL Authenticator | Y | N |

## Server

```
$ go run cmd/server/main.go
```
